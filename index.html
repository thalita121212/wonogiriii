Metodologi Penelitian

Metode & Teknik Analisis

Pendekatan ilmiah yang digunakan dalam pemetaan dan analisis spasial UMKM Kabupaten Wonogiri.

🗺️

WebGIS

Sistem Informasi Geografis berbasis web menggunakan Leaflet.js + OpenStreetMap sebagai basemap. Data koordinat dikumpulkan dari Google Places API.

📐

Nearest Neighbor Analysis

Analisis tetangga terdekat (NNI) menggunakan rumus R = D̄obs / D̄exp untuk menentukan pola distribusi spasial: mengelompok, acak, atau menyebar.

🎨

Choropleth Mapping

Peta tematik dengan pewarnaan berdasarkan kepadatan UMKM per kecamatan menggunakan skala warna hijau (rendah→tinggi).

⭕

Buffer Analysis

Analisis zona penyangga (buffer) radius 1–10 km dari destinasi wisata untuk mengukur aksesibilitas UMKM terhadap arus wisatawan.

📊

Analisis Statistik Spasial

Perhitungan mean center, standard distance, minimum/maksimum jarak NN, dan Z-score untuk mengukur signifikansi pola distribusi.

🔍

Sumber Data

Data primer: Google Places API (koordinat, nama, kategori, rating). Data sekunder: BPS Wonogiri (luas wilayah 1.795 km², data kecamatan).

Alur Penelitian

1
Pengumpulan Data

Pengambilan data koordinat UMKM & wisata Wonogiri via Google Places API, menghasilkan 86 titik dengan atribut lengkap.

2
Preprocessing Data

Klasifikasi ke 11 kategori (waduk, pantai, bukit, goa, air terjun, kuliner, kafe, oleh-oleh, kerajinan, budaya, keluarga).

3
Analisis Spasial

Perhitungan NNI, choropleth mapping, dan buffer analysis menggunakan algoritma berbasis koordinat geografis (WGS84).

4
Visualisasi WebGIS

Integrasi hasil analisis ke platform WebGIS interaktif berbasis HTML5/JavaScript untuk aksesibilitas publik.
